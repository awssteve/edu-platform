## 🚀 部署指南更新

### 环境变量配置

在 `backend/.env` 中添加：

```bash
# 智谱AI配置
ZHIPU_API_KEY=your_zhipu_api_key_here

# 日志配置
LOG_LEVEL=INFO
LOG_FILE_PATH=logs/app.log
ERROR_LOG_FILE_PATH=logs/error.log
```

### 依赖包更新

确保安装以下新依赖：

```bash
pip install pypdf python-pptx python-docx
```

或更新 `requirements.txt`：

```
pypdf2==3.0.1          # PDF解析
python-pptx==0.6.23    # PPT解析
python-docx==1.1.0     # Word解析
httpx==0.25.1          # HTTP客户端（智谱API）
```

### 数据库迁移

创建新表：

```bash
cd backend
python -c "from models.project import *; from database import engine, Base; Base.metadata.create_all(bind=engine)"
```

### 启动服务

```bash
# 后端
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 前端
cd frontend
npm run dev
```
