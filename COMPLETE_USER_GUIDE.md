## 🚀 快速启动（5分钟）

### 步骤1：启动后端服务
```bash
cd backend
python3 -m uvicorn main:app --reload --port 8000
```

**等待看到**: `INFO: Uvicorn running on http://0.0.0.0:8000`

### 步骤2：启动前端服务
```bash
cd frontend
npm run dev
```

**等待看到**: `Local: http://localhost:5173/`

### 步骤3：打开浏览器
访问：http://localhost:5173
